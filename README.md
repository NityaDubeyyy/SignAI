# Local signAi Application

A local application for bidirectional Sign Language translation, supporting translation from spoken/written languages to animated sign language, and from camera-input sign language to text/speech.

## Prerequisites

- Install [Node.js](https://nodejs.org/) (Version >= 18, npm >= 9).

## Running the Application

### 1. Install Dependencies
Run the following command in the `translate-master/` directory to install dependencies:
```bash
npm install
```

### 2. Start the Local Server
Run the local development server:
```bash
npm start
```
Once the server is running, open [http://localhost:4200/](http://localhost:4200/) in your web browser.

### 3. Build for Production
To build a production-optimized version of the application:
```bash
npm run build
```

### 4. Running Tests
To run unit and integration tests:
```bash
npm test
```
